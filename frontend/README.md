# 🚀 Login & Signup Application

## 📌 Project Description

This is a full-stack web application for user login and signup functionality.

---

## 🛠️ Tech Stack

* React.js (Frontend)
* Node.js + Express.js (Backend)
* MySQL (Database)

---

## 📁 Project Structure

```
carrierpage/
  ├── frontend/
  ├── backend/
  └── README.md
```

---

## ⚙️ Setup Instructions

### Frontend

```
cd frontend
npm install
npm start
```

### Backend

```
cd backend
npm install
node index.js
```

---

## 🗄️ Database (MySQL)

```
CREATE DATABASE user_auth;

USE user_auth;

CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  email VARCHAR(255),
  password VARCHAR(255)
);
```
